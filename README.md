## Projects

### COVID-19 Booster Vaccination Effectiveness Analysis

An in-depth analysis of whether COVID-19 booster (3rd/4th) doses contributed to reductions in severe cases and mortality, using publicly available Korean CDC data.

| Item | Detail |
|------|--------|
| **Dataset** | Korea Disease Control and Prevention Agency (KDCA) public data |
| **Methods** | Time-series analysis, correlation analysis, visualization |
| **Tools** | Python, Pandas, Matplotlib, Folium (OSM) |
| **Key Finding** | Reductions in severe cases and mortality correlated more strongly with Omicron variant characteristics than with booster dose administration |

> **Note**: Geographic visualizations use Folium (OpenStreetMap), which may not render on GitHub. Use [nbviewer](https://nbviewer.org/) for full interactive output.
